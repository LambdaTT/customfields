# lambdatt-php/customfields

A SplitPHP Framework plugin for a custom fields mechanics.

---

## Installation

Install via Composer:

```bash
composer require lambdatt-php/customfields
```

Run the Migrations:
```bash
php console migrations:apply --module=customfields
```

**PS: this can only be installed on a SplitPHP Framework project. For more information refer to: https://github.com/splitphp/core**
